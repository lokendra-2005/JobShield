# JobShield 🛡️

A web app that helps job seekers detect scam job postings and suspicious links using AI-powered analysis.

Built for our college hackathon!

## Features

- **Fake Job Detector** — paste a job description and get a risk score
- **Link Scanner** — check if a URL is safe before clicking
- **Resume Analyzer** — get ATS score and tips to improve your resume

## Tech Stack

- Frontend: React + Vite + Tailwind + Framer Motion
- Backend: Node.js + Express + MongoDB
- Auth: JWT

## Setup

```bash
# Backend
cd backend
npm install
# add a .env with MONGODB_URI and JWT_SECRET
node server.js

# Frontend
cd frontend
npm install
npm run dev
```
